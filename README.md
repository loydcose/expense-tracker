# Expense Tracker
A simple application that tracks your expenses. 

Made with [Vite](https://vitejs.dev/), [React](https://reactjs.org/) and [Tailwindcss](https://tailwindcss.com/).

## Usage
After cloning the project, install all of the dependencies.

```bash
npm install
```
Run the project.
```bash
npm run dev
```
